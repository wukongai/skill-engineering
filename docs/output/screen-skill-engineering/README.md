# Screen Skill Engineering 展示素材包

这是一份用于介绍 Skill Engineering 的对外展示素材包，采用“真实用户流程摘要 + 渲染版演示图”的形式。

## 内容

- `screen-skill-engineering-user-journey.md`：面向普通用户的创建、确认、试用和结构化维护展示稿。
- `screen-skill-engineering-real-e2e-summary.md`：真实用户实操的脱敏摘要，保留可用于营销叙事的事实和边界。
- `screen-skill-engineering-ob-material-card.md`：可复制到 OB AI 素材库的素材卡版本。
- `assets/skill-engineering-user-journey/`：4 张用户旅程演示图。
- `assets/skill-engineering-remote-e2e/`：4 张远程安装与维护演示图。

## 素材边界

- 8 张图片来自 Skill Engineering 仓库现有的演示素材，不是本轮黑屏检查图。
- 本包没有伪装成 Codex 屏幕实录，也没有生成黑屏录屏。
- 没有复制完整对话、内部线程 ID、绝对路径、凭证或工具原始输出。
- 原始来源仍保留在 Skill Engineering 仓库的 `docs/testing/` 下；本目录是可运营的输出副本。
