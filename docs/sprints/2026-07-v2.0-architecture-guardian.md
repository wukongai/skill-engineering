# Sprint：v2.0 Architecture Guardian Phase 1

状态：In Progress

日期：2026-07-16 起

## 目标

开始 2.0 开发，先建立 Blueprint/IR 的稳定契约和只读生成能力，为后续架构守护、semantic diff 和迁移计划提供共同事实层。

## 进入范围

- Blueprint 三轴：component role、execution topology、governance level；
- schema version、unknown/legacy、扩展字段和 fingerprint；
- atomic/router/orchestrator/adapter 最小 fixture；
- 1.x contract/Doctor 的只读兼容输入；
- schema validation、negative/high-risk/migration/rollback regression；
- 只读 inventory/preview，不自动修改 Skill。

## 不进入范围

- 依赖图和 semantic diff 的完整实现；
- 自动拆分、迁移、压缩、废弃和 apply；
- 云端、LLM provider、Agent Skill Hub 分发；
- 2.0 公开发布或破坏 1.x 契约。

## 风险

- Blueprint 过早固化：第一阶段保留 unknown/扩展字段，不把推断写成事实。

- 自动修改过早：所有输出先是 preview plan，不接入 apply。

## 退出条件

- Blueprint schema/version/fingerprint 已确定；
- fixture 和负例覆盖三轴、unknown、敏感信息和迁移；
- 只读 inventory 对 1.x Skill 可重复；
