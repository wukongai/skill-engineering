# Sprint：v1.0 Stable Lifecycle Contract

状态：Closed / Stable；全部门禁通过，正式 tag 与 GitHub Release 已发布

日期：2026-07-18

## 目标

把已经通过真实用户旅程验证的 0.1.x 生命周期能力，收敛为用户可以稳定安装、创建、检查、维护和回滚的 1.0 契约，并使远程默认分支与《Skill Engineering 1.0 开源与安装指南》一致。

## 进入范围

- 唯一身份 `skill-engineering` 与标准 `npx skills add` 安装入口；
- CLI、JSON schema、审批和错误边界冻结；
- wheel、Python 支持矩阵、干净环境安装；
- 会议纪要、GitHub Issue、Web 验收、研究证据包四个 Use Case；
- README、版本源、Changelog、Roadmap、Task、发布日志和测试证据对齐。
- Apache-2.0、NOTICE、citation、品牌边界和包元数据在首个稳定 tag 前冻结。

## 不进入范围

- Architecture Guardian、Blueprint inventory 和 semantic diff 的 2.0 正式能力；
- PyPI 发布、自动 Global 安装、自动 tag 或未经批准的 GitHub Release；
- 用静态 Doctor 分数宣称跨模型真实效用。
