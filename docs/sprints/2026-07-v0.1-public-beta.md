# Sprint：v0.1 Public Beta

状态：Release Candidate Verified，真实仓库已应用并获得公开发布授权。

日期：2026-07-13

## 目标

把已经可运行的 Skill Engineering 核心收敛成可供小范围用户使用的开源 Public Beta：既能快速生成从第一版开始就符合工程规范的 Skill，也能让它在后续维护中持续保持架构和质量。

## 范围

- 创建同一计划 Apply 和 postflight；
- 架构优先的快速生成与全生命周期维护统一产品表达；
- 复杂/商业 Skill 工程治理路由；
- 产品、版本、Backlog、Spec/Plan、ADR、Daily Log；
- README、贡献、安全和 CI；
- 完整本地验证与隔离 E2E。

## 不进入范围

- provider runner；
- 完整 Skill IR/compiler；
- Review Studio；
- 云端和企业功能；
- 自动 push、tag 或公开发布。

## 退出条件

- Spec 验收标准全部有实现或明确 Backlog；
- Release Gate 全部通过；
- 真实仓库应用使用同一未漂移 maintenance plan；
- 用户可从 README 完成安装、创建预览、应用和 Doctor；
- 所有未完成能力不被描述成已交付。
